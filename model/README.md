本代主要实现基于大众点评的用户推荐，初步使用了PMF,SVD和SVD++等矩阵分解方法来预测用户对商店的评分。
map_data.py主要用于实现用户映射，商店映射和用户打分映射，目的为减小处理过程中的内存占用过大，可视化不方便等问题。
split_train_test.py用于区分测试集和训练集
map文件夹为代码清洗原始数据过后的数据，数据用作后续代码的使用
PMF文件夹为使用PMF方法做评分预测
stochastic_gradient_descent_SVD文件夹为使用SVD和SVD++方法做评分预测
SocialMF为未完成代码，正在实现，主要思路是将社交网络带入推荐系统，从而使算法和现实社会更加接近，得到更加人性化的推荐。
